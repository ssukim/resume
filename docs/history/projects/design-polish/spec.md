## 목표

현재 shadcn/ui 기본 다크 테마를 그대로 사용하고 있어 시각적 개성이 없는 상태. "이 사이트만의 인상"을 만들기 위해 색상, 타이포그래피, 계층 구조, 여백, 인터랙션을 개선한다.

## 현재 문제

1. **색상** — 전체가 무채색(oklch chroma 0). 액센트 컬러 없이 CTA가 눈에 띄지 않음
2. **사이드바 프로필** — br 강제 줄바꿈, 시각 요소(사진/아바타) 없음, 이름/타이틀 위계 불명확
3. **시각적 계층** — 섹션 구분이 Separator 선에만 의존. 카드 스타일 불일관(경력 border-l-4, 프로젝트 없음)
4. **타이포그래피** — 본문 text-primary/90 가독성, 섹션 제목 크기 동일, 긴 텍스트 단락 구분 없음
5. **인터랙션** — 탭 전환 즉시 교체, 아코디언 애니메이션 없음, animate-fade-in 미사용
6. **여백** — 사이드바 패딩 과다(콘텐츠 240px), Badge 밀집 구간 여백 부족

## 범위

### 포함
- globals.css 색상 변수 수정 (액센트 컬러)
- 사이드바 프로필 영역 구조 변경
- 섹션/카드 시각적 계층 개선
- 타이포그래피 크기/불투명도 조정
- 탭 전환, 아코디언 트랜지션 추가
- 여백/패딩 조정

### 제외
- 콘텐츠(데이터) 변경
- 새 페이지/라우트 추가
- 컴포넌트 파일 분리/리팩토링
- SEO 개선 (별도 프로젝트)
