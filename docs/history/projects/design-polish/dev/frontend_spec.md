## 구현 순서

1. globals.css 액센트 컬러 적용
2. Sidebar 프로필 영역 재구성
3. 시각적 계층 (섹션 여백, 카드 border, 서브프로젝트 배경)
4. 타이포그래피 (불투명도, 섹션 제목 크기)
5. 인터랙션 (탭 fade-in, 아코디언 grid 트랜지션)
6. 여백/밀도 (사이드바 패딩, Badge gap)
