## 목표

현재 단일 컬럼 + 상단 탭 네비게이션 구조를 **왼쪽 고정 사이드바 + 오른쪽 스크롤 콘텐츠** 구조로 전환하여 모던 포트폴리오 레이아웃을 만든다.

## 현재 구조

- 단일 컬럼 (`max-w-4xl` 중앙 정렬)
- 상단 헤더: 이름, 타이틀, 연락처
- 상단 탭 네비게이션: 이력서 / 케이스 스터디
- 탭 아래 콘텐츠 영역
- 하단 푸터

## 변경 후 구조

```
┌─────────────┬──────────────────────────────┐
│  사이드바     │  메인 콘텐츠 (스크롤)         │
│  (w-72)     │  (flex-1)                    │
│             │                              │
│  프로필      │  탭 콘텐츠                    │
│  네비게이션   │  (이력서 / 케이스 스터디)      │
│  Core Skills│                              │
│             │                              │
│  ─ ─ ─ ─   │                              │
│  연락처      │  푸터                         │
│  (하단 고정) │                              │
└─────────────┴──────────────────────────────┘
```

## 사이드바 구성

| 영역 | 내용 |
|------|------|
| 프로필 | 타이틀, 이름, Last Updated |
| 네비게이션 | 이력서 / 케이스 스터디 버튼 |
| Core Skills | 주요 기술 4개 Badge |
| 연락처 (하단) | 전화, 이메일 |

## 범위

### 포함
- `page.tsx` JSX 구조 변경 (사이드바 + 메인 콘텐츠)
- 모바일 반응형 (햄버거 메뉴 → 사이드바 슬라이드인)
- 모바일 상단 고정 바 (이름 + 햄버거 버튼)
- 탭 전환 시 메인 콘텐츠 스크롤 초기화

### 제외
- 컴포넌트 파일 수정/분리
- 데이터/타입 정의 변경
- `layout.tsx` 변경
- 새로운 페이지/라우트 추가
- 다크모드

## 수정 대상 파일

| 파일 | 변경 내용 |
