| 모바일 오버레이 | `bg-black/50` | 반투명 검정 |

## 6. 인터랙션 상세

### 6-1. 네비게이션 탭 전환

| 상태 | 스타일 |
|------|--------|
| 비활성 | `text-muted-foreground`, 배경 없음 |
| 호버 | `hover:bg-sidebar-accent`, `hover:text-sidebar-accent-foreground` |
| 활성 | `bg-sidebar-accent`, `text-sidebar-accent-foreground`, `font-medium` |

- 탭 클릭 시 메인 콘텐츠 `scrollTop = 0` 초기화
- 모바일: 탭 클릭 시 사이드바 자동 닫힘

### 6-2. 모바일 사이드바 토글

| 이벤트 | 동작 |
|--------|------|
| 햄버거 클릭 | 사이드바 슬라이드인 (left: 0), 오버레이 표시 |
| X 클릭 | 사이드바 슬라이드아웃 (left: -288px), 오버레이 제거 |
| 오버레이 클릭 | 사이드바 닫힘 |
| 탭 전환 | 사이드바 닫힘 |

### 6-3. 트랜지션

| 요소 | 속성 | 값 |
|------|------|------|
| 사이드바 슬라이드 | `transition-transform` | `duration-300 ease-in-out` |
| 오버레이 페이드 | `transition-opacity` | `duration-300` |
