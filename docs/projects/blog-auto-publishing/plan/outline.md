# Claude Code로 블로그 자동 발행 시스템 만들기 — 아웃라인

## 도입
- 블로그 글은 쓰고 싶지만 발행이 번거롭다
- 작성부터 발행까지 전 과정 자동화

## 본문

### 1. 워크플로 설계
- CLAUDE.md의 "블로그 써줘" 명령어 구조
- 주제 선정 → 톤 확인 → 메타 수집 → 작성 → 발행

### 2. Playwright MCP로 폼 자동화
- DEV 환경 블로그 작성 페이지 접속
- 제목, 내용, 태그, 썸네일 자동 입력
- 미리보기 확인 후 발행

### 3. 실제 발행 시연
- conma 블로그 실제 포스트 발행 과정
- 스크린샷과 함께 단계별 설명

## 결론
- 블로그 발행 소요 시간: 1시간 → 10분
- 컨텐츠 품질에만 집중, 나머지는 AI가 처리
